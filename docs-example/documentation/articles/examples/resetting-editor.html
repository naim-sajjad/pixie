Resetting Editor

Editor can be fully reset via pixie.resetEditor() api method. This way you can fully clear the canvas, open a new image, project or blank canvas.

var pixie = new Pixie({
    image: 'http://via.placeholder.com/350x150'
});

//reset editor and open new image
pixie.resetEditor('image', 'https://my-new-image.png');

You can also reset and open editor in single pixie.resetAndOpenEditor() call.

var pixie = new Pixie({
    image: 'http://via.placeholder.com/350x150'
});

pixie.close();

pixie.resetAndOpenEditor('image', 'https://my-new-image.png');
